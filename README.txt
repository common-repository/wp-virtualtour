Easily create interactive virtual 360° tours.

== Description ==

Create 360° virtual tour for your visitors.

Key features:

*   fast and easy interface to create tours
*   Gutenberg block to easily embed tours

== Installation ==

1. Upload `wp_virtualtour` folder to the `/wp-content/plugins/` directory.
1. Activate the plugin through the 'Plugins' menu in WordPress.
1. You will find a 'WP Virtual Tour' menu entry to create tours.
1. Embed the thour into a post/page via the 'WP Virtual Tour' block
